          Calculate a new Raster by performing the map algebra specified by the Jiffle/MapCalc script.

          Parameters:
          script - The Jiffle/MapCalc script. This can be a File, a URL, or a String
          rasters - A Map of the input Rasters and their names.
          outputName - The output Raster name.
          size - The size of the output Raster
          bounds - The geographic Bounds of the output Raster
