              Create a new PostGIS Workspace with a name, host, port, schema, user, and password.

              Parameters:
              name - The database name
              host - The host name
              port - The port
              schema - The database schema
              user - The user name
              password - The password
              estimatedExtent - Whether to estimate the extent or not
              createDatabase - Whether to create the database or not
              createDatabaseParams - The database creation parameter string
