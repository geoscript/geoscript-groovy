[Groovy] Class Layer

    • geoscript.layer.Layer
    • All Implemented Interfaces and Traits:
      Renderable

      class Layer
extends java.lang.Object

      A Layer is a source of spatial data that contains a collection of Features. Most often Layers are accessed from a geoscript.workspace.Workspace but you can create an in memory Layer by simply passing a name and a geoscript.feature.Schema: 

       Schema schema = new Schema("facilities", [new Field("geom","Point", "EPSG:2927"), new Field("name","string"), new Field("address","string")])
 Layer layer = new Layer("facilities", schema)

      If, all you want to store in a Layer is geoscript.geom.Geometry, you can just pass a layer name: 

       Layer layer = new Layer("points")
 layer.add([new Point(0,0)])
 layer.add([new Point(1,1)])
