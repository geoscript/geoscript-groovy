Package: geoscript.feature

[Groovy] Class Field

    • geoscript.feature.Field

    • A Field is composed of a name and a type. A Field with a Geometry type can also contain a Projection. 

       Field f1 = new Field("name","String")
 Field f2 = new Field("geom","Point", "EPSG:2927")
